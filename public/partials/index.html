Getting started

Not sure where to go from here? Simply get started using the following steps:

  1. Start by creating a new project.
  2. Install the GraPHPizer CLI tool into your project, using composer require helmich/graphpizer-cli
  3. Create .graphizer.json configuration files in your codebase to control how GraPHPizer treats your source code during import and analysis.
  4. Import your project's source code into GraPHPizer using composer/bin/graphpizer-cli ast:import .
  5. Have a coffee.
  6. Perform the model generation step by pressing the Generate/Update Model button in the project overview.
